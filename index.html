📋Résumé exécutif

Une annonce BGP frauduleuse vise le préfixe 90.98.0.0/15 appartenant légitimement à AS3215 Orange France. L'investigation initiale avait identifié une chaîne complexe impliquant plusieurs ASN sur plusieurs continents.

Après revue technique complémentaire, les conclusions ont été affinées : la complexité apparente de l'AS path résulte en grande partie d'une falsification intentionnelle de segments du chemin BGP destinée à brouiller l'attribution.

Conclusion consolidée : AS41128 est l'origine frauduleuse. AS29802 Hivelocity est l'infrastructure réelle côté destination dans la version récente. AS199524 G-Core était le seul transit réel dans une version antérieure. Hypothèse finale : spam / usage abusif d'espace IP non blacklisté.

🎯Contexte de la correction

La révision des conclusions ne constitue pas une invalidation de l'investigation mais une amélioration de sa rigueur méthodologique. C'est le résultat normal d'une revue par les pairs.

La correction principale porte sur le rôle de plusieurs ASN dans le chemin : certains segments étaient forgés pour simuler une infrastructure multi-AS complexe, alors que la chaîne réelle est plus directe.

Pourquoi ça reste sérieux : même sur un espace inutilisé, ce type d'opération permet d'envoyer du spam depuis des IPs réputées propres, sans historique dans les blacklists. AS263692 est pré-positionné sur un second préfixe : réserve prévue quand les premières IPs seront brûlées.

🕒Chronologie

  • 25 février 2026 · 08:00 UTC Première annonce frauduleuse de 90.98.0.0/15 via AS41128. Enregistrée uniquement dans ALTDB. Statut RPKI : UNKNOWN. Aucune détection publique — 51 jours de préparation discrète.
  • 13 avril 2026 · 16:27 UTC Injection massive : 1 088 routes annoncées en 5 minutes. La route devient visible par 328/329 pairs RIPE IPv4 (99,7 %). Visibilité mondiale confirmée (Amsterdam, Londres, Frankfurt, Tokyo, São Paulo, Dubai).
  • 14 avril 2026 La version antérieure de l'AS path — où AS199524 G-Core était le seul transit réel — disparaît. L'attaquant reconstruit un AS path autour de AS29802 Hivelocity.
  • 15 avril 2026 Spamhaus interagit publiquement sur l'investigation, signalant une surveillance de l'incident.
  • 17–18 avril 2026 Revue technique incluant des échanges avec Doug Madory (Kentik). Correction et publication du rapport final.

✅Éléments confirmés

  • 90.98.0.0/15 appartient légitimement à Orange France (AS3215).
  • AS41128 est utilisé comme origine frauduleuse.
  • Le statut RPKI est UNKNOWN — aucun ROA valide n'existe pour ce préfixe, ce qui permet l'acceptation par certains réseaux.
  • AS29802 Hivelocity est le seul ASN réel côté destination dans la version récente.
  • AS199524 G-Core était le seul ASN réel dans la version précédente.
  • Hypothèse finale retenue : spam / abus réseau depuis des IPs avec aucun historique blacklist.
  • AS263692 pré-positionné sur 92.183.128.0/18 — probablement la réserve quand les premières IPs seront blacklistées.

💡Interprétation finale

L'attaque combine deux techniques complémentaires : un détournement BGP d'espace inutilisé pour obtenir des IPs avec une réputation propre, et une falsification de segments d'AS path pour compliquer l'analyse de l'incident et ralentir l'attribution.

Ce mode opératoire est cohérent avec des campagnes cherchant des IPs neuves pour de l'envoi de spam en volume, des abus réseau à faible friction ou du phishing d'IPs propres. Le fait que l'espace soit probablement inutilisé réduit la probabilité d'impact sur les abonnés Orange, mais le signal technique reste significatif et mérite action.

La présence d'un second préfixe pré-positionné (92.183.128.0/18 via AS263692) est cohérente avec un acteur qui prévoit une rotation de ses IPs quand les premières seront blacklistées.
